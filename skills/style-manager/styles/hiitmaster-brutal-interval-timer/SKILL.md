# Hiitmaster Brutal Interval Timer

This child skill captures a high-pressure interval-timer style: black stage, red-orange signal, command labels, and brutal rectangular UI.

Default page type: high-intensity training or challenge tool homepage with a live timer stage, protocol cards, and command-style CTAs.

## Read First

1. `DESIGN.md` for tokens, signature moves, and style rules.
2. `references/source-observations.md` for extraction evidence.
3. `references/code-patterns/` for reusable mechanics.
4. `preview.html` for a local token and component scan.
5. `examples/index.html` for a fresh transfer example.

## Preserve

- Full black canvas with `#ff2d00` as the dominant signal color.
- Huge tabular timer numerals and short uppercase state labels.
- Heavy uppercase Inter-like headings, tight line-height, and command copy.
- Zero-radius slabs, hard borders, and offset shadows.
- Top progress line, faint 50px grid overlay, and subtle noise.
- Red section bars, card underlines, and command badges.
- Glitch, cut, blink, and translate-up interactions used as intensity cues.

## Avoid

- Copying HIIT Master branding, product copy, app screenshots, Play Store badges, or exact page sequence.
- Soft SaaS gradients, glassmorphism, rounded cards, pastel fitness wellness styling, or lifestyle photography as the main mood.
- Turning every element red. Keep red for signal, action, timer, and state.
- Letting huge text clip on mobile. Use fixed breakpoints and test screenshots.

Use `DESIGN.md` as the design source of truth, then adapt the grammar to the user's content and stack.
