# Lonely Blue Field Notes

Dark editorial technical-journal style derived from `https://www.neurohack.blue/`.

## Files

- `DESIGN.md`: canonical design system
- `SKILL.md`: agent usage instructions
- `preview.html`: token specimens for colors, typography, and shapes
- `references/source-observations.md`: source audit trail
- `references/code-patterns/`: reusable component and section snippets
- `examples/index.html`: confirmed transfer example generated with this skill

## Examples

Confirmed applied examples demonstrate style transfer to new page types. They should not clone the source site.

## Notes

This style should feel like private field notes rather than a SaaS landing page.
