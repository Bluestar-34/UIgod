# Notion Friendly Workspace

Notion-inspired AI workspace campaign style derived from `https://www.notion.com/`.

## Files

- `DESIGN.md`: canonical design system
- `SKILL.md`: agent usage instructions
- `preview.html`: token specimens for colors, typography, and shapes
- `references/source-observations.md`: source audit trail
- `references/code-patterns/`: reusable component and section snippets
- `examples/applied/`: confirmed transfer examples generated with this skill

## Examples

No confirmed applied example has been added yet. Future examples should demonstrate style transfer to new page types and should not clone the source site.

## Notes

The style's strongest signals are a dark midnight campaign hero, white product window, sticker-like agents, hand-drawn blue tracks, and warm Notion-like bento sections.
