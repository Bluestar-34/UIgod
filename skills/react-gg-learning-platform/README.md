# react.gg — 学习平台风格

交互式学习平台深色主题风格，源自 [react.gg](https://react.gg/)。

## 风格概要
- **背景**: 煤黑 (#0f0d0e) + 炭色网格纹理
- **文字**: 暖米色 (#f9f4da) · 强调: 金黄 (#fcba28)
- **字体**: Paytone One (标题) + Outfit (正文) + Fira Code (代码)
- **形状**: 药丸按钮 + 页面外框 + 卡片圆角
- **氛围**: 专业 · 现代 · 温暖 · 互动
