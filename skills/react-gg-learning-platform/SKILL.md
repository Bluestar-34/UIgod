# react.gg — 学习平台风格

交互式学习平台的深色主题视觉语言。

默认页面类型：学习平台首页。

## 快速开始

1. `DESIGN.md` — 设计系统
2. `references/` — CSS提取 + 代码模式
3. `examples/index.html` — 风格转移示例
4. `preview.html` — 令牌标本

## 不可协商元素

- **煤黑(#0f0d0e)背景 + 炭色网格纹理**
- **暖米色(#f9f4da)文字 — 非纯白**
- **金色(#fcba28)主强调**
- **Paytone One 大写标题**
- **药丸按钮 (9999px) + 薄阴影 (2px 2px 0)**
- **页面外框 (1.3rem 边框)**
- **链接 hover 反转 (黄底黑字)**
