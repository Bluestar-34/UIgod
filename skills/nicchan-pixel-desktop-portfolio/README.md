# Nicchan Pixel Desktop Portfolio

Reusable UIgod child skill derived from [nicchan.me](https://www.nicchan.me/).

It captures an accessible pixel desktop portfolio style: violet pixel-art wallpaper, left icon rail, bottom taskbar, W95-style windows, hard borders, inset button shadows, and light/dark preference support.

## Files

- `DESIGN.md` - canonical design system and guardrails
- `SKILL.md` - operational instructions for agents
- `preview.html` - local token and component specimen
- `references/source-observations.md` - CSS extraction, screenshots, provenance
- `references/code-patterns/` - reusable snippets for nav, windows, taskbar, pixel patterns
- `examples/index.html` - fresh portfolio reference site in the same style family

Open `preview.html` or `examples/index.html` directly in a browser. No build step is required.
