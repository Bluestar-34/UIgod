# Applied Example

`index.html` is a fresh reference site for a fictional independent front-end developer named Mira Vale.

It demonstrates style transfer only:

- desktop icon navigation
- pixel-art wallpaper generated in CSS
- window panels for intro, work, process, notes, and contact
- bottom taskbar
- W95-style typography, borders, and buttons

It intentionally does not copy Nic Chan's portrait, client logos, posts, or exact page structure.
