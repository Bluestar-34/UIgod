## Observed Facts

- Current hero headline: "Build and deploy on the AI Cloud."
- The page is black-canvas first with a white Vercel wordmark, muted nav links, and compact auth actions.
- Primary CTA is a white pill with black text. Secondary CTA is a dark pill with light text.
- A centered event strip appears above the hero with a blue label and white ticket pill.
- The hero visual sits inside a large hairline grid frame.
- The main visual uses a luminous triangular mark and spectral rays in blue, green, amber, and red.
- Section architecture uses thin borders and grid divisions rather than elevated cards.
- Product messaging is developer-focused: AI apps, web apps, commerce, multi-tenant platforms, infrastructure, security, scale.
- Typography is Geist-like: precise, sans-serif, tight, white headings with muted grey body copy.
- Interface fragments are technical: terminal panels, browser frames, workflow nodes, deployment diagrams, tabs, and status chips.

## Why This Reference Is Useful

Vercel is a strong test for UIgod because the visual language is specific but deceptively minimal.

The risk is generating a generic dark SaaS page. The useful extraction must preserve:

- black-and-white discipline
- grid/framing system
- triangle geometry
- spectral light as the main color behavior
- compact developer UI fragments
- infrastructure-module composition
