# Vercel AI Cloud Minimal

Vercel-inspired black-canvas developer infrastructure style derived from `https://vercel.com/`.

## Files

- `DESIGN.md`: canonical design system
- `SKILL.md`: agent usage instructions
- `preview.html`: token specimens for colors, typography, and shapes
- `references/source-observations.md`: source audit trail
- `references/source-screenshots-or-notes/`: optional source screenshots and notes
- `references/code-patterns/`: reusable component fragments
- `examples/applied/`: confirmed transfer examples generated with this skill

## Examples

Confirmed applied examples demonstrate style transfer to new page types. They should not clone the source site.
