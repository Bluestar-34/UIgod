# Studio Brot — Playful Agency

A child skill capturing the loud, bold, irreverent German creative agency style of [studio brot](https://www.studiobrot.de/) from Stuttgart.

## What This Is

This is a reusable web style skill generated by UIgod. It preserves the visual identity of a confident, personality-driven advertising agency: alternating pink (#FFBEF1) and red (#ff2a00) section backgrounds, custom FKScreamer display typography at extreme sizes, thick black borders (3.5–7px), pill-shaped CTAs with bold drop shadows and press-down hover animations, black-background logo marquees, and self-aware copy.
