# Studio Brot — Playful Agency

This child skill captures the loud, irreverent, and unmistakably bold German advertising agency style of studio brot (Stuttgart).

It is designed for web work that should feel like a confident, personality-driven creative agency: extreme typography, pink/red section alternation, chunky pill buttons with shadows, rough black borders, and self-aware copy.

Default representative page type: a creative agency homepage (red hero → showreel → intro → black marquee → pink case studies → white services → statement → pink contact → newsletter → red footer). When the user does not specify a page type, generate this one.

## Use When

Use this skill for:

- creative / advertising agency websites
- bold personal portfolio or studio sites
- brand landing pages that need attitude
- any project where subtle is the wrong answer

## Read First

- `DESIGN.md` for the canonical design system (start here)
- `references/code-patterns/` for reusable component implementations
- `examples/applied/` for confirmed transfer examples
- `preview.html` for a quick token scan

## Style Priority

Preserve these non-negotiable elements:

- **Red/pink section alternation** — sections switch boldly between red bg, pink bg, white bg, and black bg
- **Thick black borders** (3.5–7px) on all interactive elements
- **Pill shapes** (45px border-radius) for buttons and inputs
- **Bold box-shadows** (6–12px) with press-down hover animation
- **FKScreamer or equivalent bold condensed display font** at extreme sizes
- **Self-aware, witty copy** — the brand has a sense of humor

Framework choice comes after the style logic is understood.

Use `DESIGN.md` as the single design truth source.

## Expected Output

A strong result should feel unmistakably bold:

- the first section hits with brand red background and a massive FKScreamer headline
- pink and red sections alternate, keeping energy high throughout
- every button is a pill with a thick border and a shadow that presses down on hover
- case studies sit on pink background with autoplay video
- client logos scroll on black
- the footer is brand red, not neutral
- copy is witty and confident — never corporate-stiff

## Guardrails

- Keep the red/pink color duet — no additional accent colors.
- Every interactive element must be pill-shaped with a thick border and shadow.
- Use extreme display typography (100–400px range) for headlines.
- Alternate section backgrounds — never leave them all white.
- The footer MUST be brand red background.
- Preserve the press-down hover effect on all buttons.
