# Namesake Soft Civic Guides

This child skill captures a soft, accessible civic-guide style: public-service clarity with handmade warmth.

Default page type: public-benefit guide or resource homepage.

## Read First

1. `DESIGN.md` for tokens, signature moves, and style rules.
2. `references/source-observations.md` for extraction evidence.
3. `references/code-patterns/` for reusable mechanics.
4. `preview.html` for a visual token/component scan.
5. `examples/index.html` for a fresh transfer example.

## Preserve

- Pastel single-color canvas with sparse black speckles.
- Black ink as the main text, button, border, icon, and footer color.
- Atkinson Hyperlegible Soft-like typography with heavy, friendly display weight.
- Very small radius: 4-5px controls, large circular annotation only when hand-drawn.
- Inverse primary buttons: black fill, pastel text, 3px border behavior.
- Thin civic banners and status panels that feel useful before decorative.
- Map, form, guide, checklist, or resource-table patterns for civic information.

## Avoid

- Generic purple SaaS gradients.
- Glass, glow, blur, soft card shadows, or over-rounded app UI.
- Heavy decorative illustration that overwhelms plain-language guidance.
- Copying Namesake text, map data, sponsors, logo, passport art, or snail asset.
- Replacing the calm civic tone with startup hype.

Adapt the style to the user's content and framework, but keep the signature moves visible.
