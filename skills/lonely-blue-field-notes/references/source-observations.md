## Observed Facts

- Built with Astro.
- Default theme on load is dark.
- Home screen uses a two-column gate: profile rail plus manifesto content.
- Primary content section uses stream-like rows with image, index, text, and arrow.
- A repeated-text signal band sits between content areas.
- A floating vertical scroll dock acts as a utility instrument.
- The site supports theme switching, but the dark mode is the primary mood.
- Tokens captured from the stylesheet include near-black paper surfaces, warm text, gold emphasis, and blue/green/ember signal accents.

## Why This Reference Is Useful

This site is a good child-skill seed because it has:

- a strong point of view
- distinctive typography
- specific structural behavior
- enough custom utility detail to avoid reading as a generic theme

It is not just "dark blog design." It has an authored identity system.
